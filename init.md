# Genesis World - Project Initialization

This document tracks the initial project setup.

## Initialized Components

- Monorepo structure with Turborepo
- Client app (Vite + Three.js)
- Server app (Fastify + Socket.io)
- Shared package with types and utilities
- Game theory behavior system
- Docker compose for local development

## Next Steps

1. Run `pnpm install` to install dependencies
2. Start Docker containers: `docker compose up -d`
3. Run `pnpm dev` to start development servers
4. Open http://localhost:5173 in browser
